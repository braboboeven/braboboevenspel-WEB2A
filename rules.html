spelregels

1. het spel is in groepjes behalve als je een kleine klas hebt kan het individueel dus iedereen apart.
2. ieder groepje speelt op 1 laptop.
3. voor de docent is er een dashboard die alle scores bij houdt van de leerlingen.
4. per verdachte moet een groepje 5 SQL statements invoeren
per goede query kan je maximaal 1000 dollar verdienen. als
je een fout hebt gemaakt door geen hoofdletters te gebruiken
of een puntkomma bent vergeten krijg je 500 dollar. per verdachte kun je dus maximaal 5000 punten verdienen.
5. Het spel eindigt als je het snelste bent met je groepje die 10000 punten heeft verdiend dan heb je gewonnen. De docent kan ook zien hoever iedereen is.
6. het zijn altijd open vragen.
